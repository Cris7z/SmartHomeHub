# 一\.接线

```Plain Text
ESP32 3V3  -> 面包板左红线   = 3.3V
ESP32 GND  -> 面包板左蓝线   = GND

外部电源 5V+ -> 面包板右红线 = 5V
外部电源 5V- -> 面包板右蓝线 = GND

左蓝线 GND 和 右蓝线 GND 相连

AHT20、红外接收、按键：接 3V3
LD2410C：接 5V
WS2812B：用外部 5V 电源，必须和 ESP32 GND 共地
ILI9341：优先 3V3；如果模块明确支持 5V，VCC 可接 5V，但信号线仍然接 ESP32 的 GPIO
ESP32 GPIO 不能接 5V 输出信号

AHT20温湿度传感器:
SDA -> GPIO4
SCL -> GPIO5
VCC -> 3V3
GND -> GND

ILI9341 屏幕:
SCK  -> GPIO12
MOSI -> GPIO11
CS   -> GPIO10
DC   -> GPIO9
RST  -> GPIO8
BL   -> GPIO7
VCC  -> 3V3 或模块支持的 5V
GND  -> GND

LD2410C人体存在雷达:
OUT -> GPIO16
VCC -> 5V
GND -> GND

WS2812B 灯带:
DIN -> GPIO18
5V  -> 外部 5V 电源
GND -> 外部电源 GND，并且和 ESP32 GND 共地

有源蜂鸣器:
VCC3.3V，如果声音太小再接 5V
GNDGND
IO  -> GPIO6

麦克风声音模块:
VDD 3.3V
GND GND
SCK GPIO15
WS GPIO19
SD GPIO20
L/R GND

红外发射:
DAT -> GPIO14
VCC       -> 3V3
GND       -> GND

红外接收:
DAT/OUT/S -> GPIO17
VCC       -> 3V3
GND       -> GND

模式按键:
按键1 OUT -> GPIO1
按键2 OUT -> GPIO2
按键3 OUT -> GPIO3
按键4 OUT -> GPIO13
按键5 OUT -> GPIO21

所有按键 VCC -> 3.3V
所有按键 GND -> GND
```

# 三\.软件设置

1. 顶部菜单点：

```Plain Text
Tools > Board > esp32 > ESP32S3 Dev Module
```

如果没有 `esp32`，说明 IDE 2\.3\.8 没读到之前装好的 ESP32 环境，需要我再帮你配一遍 IDE 的 Board Manager。

2. 再选端口：

```Plain Text
Tools > Port > COM10
```

3. 再确认：

```Plain Text
Tools > USB CDC On Boot > Enabled
```

4. 然后点左上角 `✓` 编译，或者 `→` 上传。

## **三、自动控制逻辑**

1. **有人模式**

    - LD2410C 检测到人：

```Plain Text
presence = 1
```

- 屏幕显示：

```Plain Text
ROOM: OCCUPIED
```

- 灯带亮。

2. **无人模式**

    - 一段时间无人后：

```Plain Text
ROOM: EMPTY
SEC: ARMED
```

- 系统自动进入安防状态。

3. **声音报警**

    - 安防状态下，麦克风检测到大声音：

```Plain Text
sound = 1
alarm = 1
```

- 蜂鸣器响。

- 屏幕显示：

```Plain Text
ALARM: NOISE!
```

- 灯带变红。

4. **温度过高空调模拟**

    - 温度超过 `28°C`：

```Plain Text
AC: COOLING
```

- 红外发射模块发送一段演示红外信号。

- 串口显示：

```Plain Text
[AC] Demo IR cooling command sent
```

5. **红外接收**

    - 遥控器对着红外接收模块按键：

```Plain Text
IR: RX
```

- 串口输出：

```Plain Text
ir=1
```

# **六、项目最终验收指标**

你可以按这个写项目指标：

1. 系统能通过屏幕实时显示室内环境和设备状态。

2. 系统能通过人体存在传感器判断有人/无人。

3. 系统能在无人安防状态下检测异常声音并触发蜂鸣器报警。

4. 系统能根据温度状态模拟红外空调控制。

5. 系统支持 5 个实体按键进行模式、灯光、空调、蜂鸣器测试和报警清除。

6. 系统能通过 RGB 灯带反馈有人、报警等状态。

7. 系统能通过串口输出全部传感器和执行器状态，方便调试。
